Graph partitioning library - Maven project installation
1. Open console in the main project folder graph_partitioning_methods
2. Write the following command: mvn install
3. The project is ready to be used. Open the project in Eclipse to examine or run the examples with the following command:
	mvn exec:java -Dexec.mainClass=examples.Start
To package the project into jar library use the following command:
	mvn package
After that the jar file will be located into the project folder target.
